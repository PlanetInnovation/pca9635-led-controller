# Pca9635

MicroPython driver for PCA9635 16-channel I2C LED controller with PWM

## Next Steps for Developers

**Remove or refine this section once you've completed the one-time setup tasks below.**

### 1. Repository Setup
```bash
# Initialize git repository and add GitLab remote
git init
git symbolic-ref HEAD refs/heads/main
git remote add origin git@gitlab.pi.planetinnovation.com.au:[your-group]/micropython-pca9635.git

# Initialize submodules and install pre-commit
make init
```

### 2. GitLab CI Compliance Setup
For GitLab CI to work properly:
1. Go to project **Settings > Access Tokens**
2. Create token named `COMPLIANCE_CHECK` with:
   - Role: **Maintainer**
   - Scopes: **read_api** only
3. Go to **Settings > CI/CD > Variables**
4. Add variable `COMPLIANCE_CHECK` with the token value
5. Set as **Masked** (check box), leave **Protected** unchecked

### 3. Development Workflow
```bash
# Run tests (preferred method)
make tests

# Run all linting and checks
make checks

# View all available commands
make help
```

### 4. Start Development
- Implement your library in `micropython_pca9635.py`
- Add tests to `test/test_micropython_pca9635.py`
- Update `manifest.py` with any runtime dependencies
- Add usage examples to `examples/`
